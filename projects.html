NLP Sentiment Analysis Project

For this project in my CS 348 course, we were given 10,000 movie reviews, then asked to create a model that accurately
classified the reviews as good or bad. In order to do so, I first cleaned the input data for the mode. I then utilized Bayesian
classifiers, cross validation, and feature extraction to accurately discriminate between positive and negative movie reviews.
